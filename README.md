# 我的数据小馆 | pk

一个以 **代码、光影、K线** 为主线的个人主页项目。页面围绕工科生身份展开，将智能感知工程学习经历、摄影作品与投资研究兴趣组织为一个可交互的静态网页。

在线访问：https://liangyi-ovo.github.io/my-data-cafe/

## 项目简介

本项目是一个纯静态个人主页，主题为：

> 代码、光影、K线之间 —— 一个工科生的数据小馆

页面通过深色冷色调、科技感视觉元素和卡片式交互，展示个人简介、摄影作品、投资研究方法和智能感知相关项目经历。整体设计目标是将理工科背景、影像表达和数据分析兴趣统一到一个完整的网页叙事中。

## 页面结构

项目包含三个主要区域：

1. **首页主题区**

   * 展示页面主标题与副标题。
   * 使用代码流、K线元素和相机视觉元素强化“代码、摄影、投资”的主题。

2. **自我介绍区**

   * 展示昵称、学校、专业和年级信息。
   * 通过标签说明个人方向：智能感知、风光摄影、价值投资。
   * 加入可点击的宠物木鱼彩蛋。

3. **数据小馆区**

   * 摄影作品集：展示个人摄影作品与旅行足迹。
   * 投资研究：展示选股策略、行业分布和交互式说明。
   * 智能感知：展示项目关键词词云和项目经历。

## 功能特性

* 原生 HTML / CSS / JavaScript 实现，无需后端服务。
* 响应式布局，适配桌面端和移动端浏览。
* 固定导航栏与页面滚动定位。
* 摄影作品卡片、详情弹窗与图片预览功能。
* 投资研究板块包含行业分布条形图。
* 智能感知板块包含项目关键词词云。
* 宠物木鱼支持点击反馈、气泡吐槽、计数反馈和能量波纹特效。

## 技术栈

* HTML5
* CSS3
* JavaScript
* GitHub Pages

项目没有使用前端框架、构建工具或后端接口，因此可以直接通过浏览器打开 `index.html` 运行。

其中：

* `assets/`：页面所需图片资源。
* `assets/previews/`：摄影作品预览图，用于降低首屏和图库加载压力。

## 本地运行

直接双击打开：

```text
index.html
```

或者在项目目录下启动本地静态服务器：

```bash
python -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000
```


## 设计思路

页面采用深色冷色调作为基础风格，用蓝色高光、玻璃拟态面板、网格背景和动态数据元素营造轻科技感。内容上以“工科、摄影、投资”三个关键词展开：

* **工科**：通过智能感知项目、视觉算法关键词和代码流体现。
* **摄影**：通过作品集、地图和影像卡片体现。
* **投资**：通过选股策略、行业分布和 K 线元素体现。

将个人兴趣和能力压缩成一个可浏览、可交互、可展示的个人数据页面。

## 后续优化方向

* 压缩大尺寸图片，提升公网加载速度。
* 将 CSS 和 JavaScript 拆分为独立文件，便于维护。
* 增加更多数据可视化组件。
* 为摄影作品增加拍摄地点、参数和作品说明。
* 为投资研究板块增加更完整的研究框架或案例展示。
